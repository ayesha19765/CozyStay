## **Project Description**  
A **modern Alpine Hotel Booking Website** designed to provide users with an immersive, seamless experience while showcasing the natural beauty of the Alps. This project focuses on creating a visually stunning, user-friendly platform that highlights the hotel's amenities, services, and picturesque location.

---

## **Overview**  
The CozyStay website embodies elegance and functionality, combining Alpine aesthetics with responsive, intuitive design. Its goal is to attract potential guests by presenting all the information needed to plan and book a dream stay in the Alps.  

### **Key Features**  

1. **Homepage**  
   - Captivating hero image or video of the Alps.  
   - Welcoming headline with a summary of the hotelâ€™s unique selling points.  
   - Quick links for room booking, exploring amenities, and viewing the photo gallery.  

2. **Room Booking**  
   - User-friendly interface with a calendar for check-in/out date selection.  
   - Detailed room descriptions including type, price, and amenities.  

3. **Photo Gallery**  
   - High-resolution images of rooms, scenic views, and hotel spaces.  
   - Interactive gallery with filters by room types and seasons.  

4. **Amenities & Services**  
   - Dedicated sections for spa, dining, outdoor activities, and other offerings.  
   - High-quality images and concise descriptions.  

5. **Dining**  
   - Menu details, opening hours, and online reservation options.  

6. **Location & Directions**  
   - Map highlighting the hotelâ€™s location in the Alps, along with directions.  

7. **Guest Reviews & Testimonials**  
   - Section showcasing ratings, reviews, and feedback from previous guests.  

8. **Contact & Support**  
   - Contact information, phone, email, and an easy-to-use contact form.  

---

## **Design Principles**  

1. **Alpine Aesthetic**  
   - Stunning images of the Alps integrated throughout the website.  
   - Elegant and minimalist design to reflect sophistication.  

2. **Responsive Design**  
   - Fully optimized for mobile, tablet, and desktop screens.  

3. **User-Friendly Navigation**  
   - Clear menus and intuitive structure for effortless browsing.  

4. **Typography**  
   - Readable fonts complementing the aesthetic.  

5. **Visual Hierarchy**  
   - Highlighting booking options, featured amenities, and scenic visuals.  

6. **Performance Optimization**  
   - Fast loading speeds for a smooth user experience.  

---

## **Technology Stack**  

- **Frontend:** HTML, CSS, JavaScript.  
- **Backend:** Node.js with Express.  
- **Database:** MongoDB for storing booking information.  

---

## **Implemented Features**  

### **Frontend**  
- Elegant, user-friendly UI with high-quality visuals.  
- Responsive design to ensure compatibility across devices.  

### **Backend**  
- API endpoints for CRUD operations.  
- Integration of secure login/logout routes.  
- Fast and optimized data retrieval with MongoDB.  

---

## **Packages Used**  

- `bcrypt`: Password hashing for secure user authentication.  
- `cookie-parser`: Simplified handling of cookies.  
- `dotenv`: Environment variable management.  
- `express`: Backend framework for building APIs.  
- `express-async-handler`: Simplifies error handling in async routes.  
- `jsonwebtoken`: For secure token-based authentication.  
- `mongodb` & `mongoose`: Efficient database management.  

---

## **How to Run Locally**  

1. Clone the repository:  
   ```bash  
   git clone <repository-link>  
   ```  
2. Install dependencies:  
   ```bash  
   npm install  
   ```  
3. Start the application:  
   ```bash  
   npm start  
   ```
